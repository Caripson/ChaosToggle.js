What's Inside

Everything you need to turn any website into a prank playground. Click any feature to read more.

34 Effects

BSOD, matrix rain, gravity, fake Windows Update, Clippy, screen crack, virus scan, and dozens more.

  • Overlays: Glitch, noise, flash, VHS distortion, CRT shutdown, screen crack, RGB split, screen melt, pixel dissolve
  • Pranks: BSOD, fake Windows Update, fake terminal, Clippy, fake virus scan, fake Chrome crash
  • DOM manipulation: Gravity (elements fall), element shuffle, element scatter, text scramble, tiny/giant mode
  • Cursor effects: Magnetic cursor, cursor chaos (rainbow trail), cursor drift, auto-typo
  • Interaction: Delayed clicks, inverted scroll, screen flip, drunk mode, popups
  • Visual: Shake (multiple styles), zoom flicker, confetti (multiple particle types), fake error state

Every effect auto-cleans after its duration. No permanent side effects.

17 Themes

Holiday packs, office pranks, hacker retro, apocalypse, drunk mode, and jumpscare. One click each.

  • Holiday: Easter, Halloween, Christmas, New Year, 4th of July, Thanksgiving, Valentine's Day, Birthday
  • Retail: Black Friday, Cyber Monday
  • Vibe: Office, Hacker, Retro, Apocalypse, Drunk, Jumpscare

Each theme brings its own palette, particle type, overlay, popup text, animation style, and behaviour modifiers. Mix-and-match with themeOverrides.

Control Panel

Draggable glass UI to tweak intensity, duration, and effects live. No build step needed.

  • Open with ChaosToggle.openPanel() or the Shift+P shortcut
  • Sliders for intensity, duration, and probability
  • Per-effect enable/disable toggles
  • Theme picker with instant preview
  • Quick action buttons for all modes
  • Export/import settings as JSON

The panel runs entirely in a Shadow DOM so it never interferes with your page styles.

Plugin System

Ship custom effects and themes as ChaosPlugin packages. Hook into engine events.

  • Register effects: ChaosToggle.registerEffect({ id, name, apply })
  • Register themes: ChaosToggle.registerTheme(name, config)
  • Compose sequences: ChaosToggle.compose('intro', [...])
  • Listen to events: ChaosToggle.on('beforeTrigger', fn)
  • Bundle as a ChaosPlugin with { name, effects, themes, install }

Plugins install via ChaosToggle.use(plugin) and automatically register their effects and themes.

TypeScript-first

Full type coverage for settings, themes, effects, and the public API. Great DX.

  • Every API method has typed params and return values
  • Exported types: ChaosToggleAPI, ChaosSettings, ChaosEffect, ChaosPlugin, ThemeProfile, ThemeBuilder
  • Theme builder with fluent API: ChaosToggle.buildTheme('myTheme').palette(...).effects(...).register()
  • Auto-complete in VS Code, WebStorm, Cursor, etc.

Zero Dependencies

One script tag or ESM import. Self-contained runtime with inlined styles.

  • CDN: <script src="https://cdn.jsdelivr.net/npm/chaos-toggle/dist/chaos-toggle.min.js">
  • ESM: import ChaosToggle from 'chaos-toggle'
  • CJS: const { ChaosToggle } = require('chaos-toggle')
  • No React, Vue, or Angular required — works with everything
  • Injects its own CSS at runtime — zero external stylesheets
