Quick Examples

Copy, paste, deploy, deny everything.

Run a specific effect

ChaosToggle.runEffect('matrixRain');
ChaosToggle.runEffect('bsod');
ChaosToggle.runEffect('gravity');

Apply a theme

ChaosToggle.runTheme('hacker');
ChaosToggle.runTheme('office');
ChaosToggle.runTheme('apocalypse');

Run a mode

ChaosToggle.runMode('nuclear');
ChaosToggle.runMode('panic');
ChaosToggle.runMode('celebration');

Compose a sequence

ChaosToggle.compose('ultimate-prank', [
  { effect: 'screenCrack', delay: 0 },
  { effect: 'bsod', delay: 800 },
  { effect: 'fakeVirusScan', delay: 4000 },
]);
ChaosToggle.runComposition('ultimate-prank');

Open the control panel

ChaosToggle.openPanel();

Configure settings

ChaosToggle.init({
  intensity: 0.7,
  duration: 3000,
  theme: 'hacker',
  effects: {
    bsod: true,
    matrixRain: true,
    gravity: false,
  },
});

What's Inside

Six pillars of professional-grade mayhem. Click any card to expand.

34 Effects

BSOD, matrix rain, gravity, fake Windows Update, Clippy, screen crack, virus scan, and dozens more.

  • Overlays: Glitch, noise, flash, VHS distortion, CRT shutdown, screen crack, RGB split, screen melt, pixel dissolve
  • Pranks: BSOD, fake Windows Update, fake terminal, Clippy, fake virus scan, fake Chrome crash
  • DOM: Gravity, element shuffle, element scatter, text scramble, tiny/giant mode
  • Cursor: Magnetic cursor, cursor chaos, cursor drift, auto-typo
  • Interaction: Delayed clicks, inverted scroll, screen flip, drunk mode, popups

Every effect auto-cleans after its duration. No permanent side effects.

17 Themes

Holiday packs, office pranks, hacker retro, apocalypse, drunk mode, and jumpscare.

  • Holiday: Easter, Halloween, Christmas, New Year, 4th of July, Thanksgiving, Valentine's Day, Birthday
  • Retail: Black Friday, Cyber Monday
  • Vibe: Office, Hacker, Retro, Apocalypse, Drunk, Jumpscare

Each theme brings its own palette, particles, overlay, popups, animation style, and behavior modifiers. Mix with themeOverrides.

Control Panel

Draggable glass UI to tweak intensity, duration, and effects live.

  • Open with ChaosToggle.openPanel() or Shift+P
  • Sliders for intensity, duration, and probability
  • Per-effect enable/disable toggles
  • Theme picker with instant preview
  • Export/import settings as JSON

Runs in a Shadow DOM -- never interferes with your page styles.

Plugin System

Ship custom effects and themes as ChaosPlugin packages.

  • ChaosToggle.registerEffect({ id, name, apply })
  • ChaosToggle.registerTheme(name, config)
  • ChaosToggle.compose('intro', [...])
  • ChaosToggle.on('beforeTrigger', fn)

Install via ChaosToggle.use(plugin).

TypeScript-first

Full type coverage. Great DX and autocomplete everywhere.

  • Every API method typed with params and return values
  • Exported: ChaosToggleAPI, ChaosSettings, ChaosEffect, ChaosPlugin, ThemeProfile, ThemeBuilder
  • Fluent theme builder: .palette(...).effects(...).register()

Zero Dependencies

One script tag or ESM import. Self-contained with inlined styles.

  • CDN, ESM, or CJS -- works with everything
  • No React, Vue, or Angular required
  • Injects its own CSS at runtime

API Reference

The serious part. (Just kidding, it's still chaos.)

MethodDescription
init(config?)Initialize with optional config
trigger()Fire all enabled effects
reset()Clear all active effects
destroy()Full teardown
updateSettings(config)Merge new settings at runtime
enable() / disable()Toggle on/off
getSettings()Get current settings copy
runEffect(id)Run a single effect by ID
listEffects()List all registered effect IDs
registerEffect(effect)Add a custom effect object
runMode(name)Run a preset mode
listModes()List available mode names
runTheme(name)Apply and trigger a theme
setTheme(name)Set theme without triggering
listThemes()List registered theme names
buildTheme(name)Start fluent theme builder
registerTheme(name, config)Register a theme object
compose(name, steps)Define a timed effect sequence
runComposition(name)Run a composed sequence
use(plugin)Install a ChaosPlugin
on(event, handler)Listen for engine events
openPanel() / closePanel() / togglePanel()Control the built-in panel
registerShortcut(combo, action)Add a keyboard shortcut

Theme Builder & Plugins

Build your own chaos. Ship it to your friends.

Fluent Theme Builder

ChaosToggle.buildTheme('office-prank')
  .palette('#0078d7', '#ffb900', '#f0f0f0', '#333')
  .effects('bsod', 'fakeUpdate', 'clippy')
  .particles('confetti', { density: 0.3 })
  .popup({
    title: 'IT Department',
    message: 'Your PC has been compromised.'
  })
  .timing({ durationMultiplier: 2 })
  .register();

ChaosToggle.runTheme('office-prank');

Plugin System

const myPlugin = {
  id: 'chaos-plugin-office',
  name: 'Office Pranks Pack',
  version: '1.0.0',
  effects: [myCustomEffect],
  themes: [{ id: 'corporate', config: {...} }],
  install(engine) {
    engine.registerShortcut('Shift+B', 'runEffect:bsod');
  },
};

ChaosToggle.use(myPlugin);
